元素的宽高

1. 块级元素

宽度自适应，高度由内部文档流元素的总和决定
可设置 padding 和 margin

2. 内联元素

宽度和高度由 content 决定
设置 padding 和 margin 无效

3. 内联块级元素

宽度和高度由内部文档流的总和决定
可设置 padding 和 margin
兼备了块级元素和内联元素的特性

4. 水平居中

块级元素：设置它自己 margin:auto
内联元素：设置其父元素 text-align:center

5. 垂直居中

块级元素：设置其父元素的 padding（或者做 flex 布局）
内联元素：设置其父元素的 padding（或者做 flex 布局）
